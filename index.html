Sobre Mim

Sempre fui do tipo que, quando não encontra uma solução, cria uma. Antes mesmo de seguir carreira na programação, já desenvolvia scripts e modificações para facilitar meu dia a dia, principalmente em jogos, meu maior hobby. Tenho prazer em transformar ideias em realidade, seja criando ferramentas web, modificando jogos para expandir possibilidades ou automatizando tarefas com Python e C++. Para mim, a computação é mais do que uma profissão — é um meio de integrar criatividade e tecnologia na rotina.

Trabalhar com o que gosto significa que cada desafio não é tempo perdido, mas uma oportunidade de evolução. Cada aprendizado tem um propósito, e cada conquista alimenta uma nova ambição.

Infraestrutura

Cloud Computing

Tenho grande interesse e venho desenvolvendo minhas habilidades em Cloud Computing, com foco na plataforma AWS. Minha introdução à AWS foi através de um projeto de integração com EC2, onde criei uma solução para comunicação remota entre dispositivos via terminal, independentemente da rede Wi-Fi. O sucesso dessa implementação me incentivou a explorar mais a AWS e aplicá-la em outros projetos, mesmo quando não era um requisito inicial.

Posteriormente, apliquei esses conhecimentos em um projeto mobile de e-commerce em tempo real. Após o AWS Cloud Practitioner Essentials, minha visão sobre arquitetura em nuvem se ampliou — aprendi a gerenciar recursos com eficiência e explorar conceitos de otimização de custos.

Atualmente utilizo Lambda (serverless), S3, DynamoDB (NoSQL) e API Gateway. Vejo a convergência entre análise, desenvolvimento e gestão de dados na nuvem como uma área extremamente promissora.

Automação

Script

Scripts tornaram-se parte fundamental da minha rotina. Aplico-os para resolver problemas práticos, explorar padrões de dados e desenvolver soluções automatizadas em diversos cenários — profissionais e pessoais.

Um dos projetos mais impactantes foi automatizar atualizações de arquivos e pastas para The Elder Scrolls V: Skyrim Anniversary Edition: renomear pastas, atualizar arquivos .txt e gerar JSONs com configurações específicas — tudo para compatibilizar mods.

O projeto envolvia mais de 2.000 pastas com dezenas de arquivos cada. Em menos de 30 minutos, o script automatizou o que levaria dias manualmente. Compartilhei a solução com a comunidade e beneficiei milhares de jogadores.
